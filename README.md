# Fruta & Fruto

Este projeto foi uma tarefa do curso de Front-End da Alura, consiste na criação de um site de uma loja de alimentos.

## Tecnologias Utilizadas
* HTML
* CSS
* Bootstrap

## Conteúdos abordados
* Navbar
* Carrossel
* Cards
* modal
* Display flex e Grid
* Responsividade
